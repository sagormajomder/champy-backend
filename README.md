# Champy - Challenge Yourself, Conquer the World

Champy is a dynamic contest platform designed to foster competition and skill development. It offers a user-friendly interface for participants to join various contests, track their progress on global leaderboards, and engage with a community of like-minded individuals. Built with modern web technologies, Champy ensures a seamless and responsive experience for all users.

## Run it Locally

Please follow the below instructions to run this project in your machine:

1. Clone this repository

   ```sh
   git clone https://github.com/sagormajomder/champy-backend.git
   ```

2. Open the directory "champy-backend" into visual studio code
3. Open Terminal and run `npm i` to install all dependencies
4. Set up environment variables:

   Create a .env file in the root directory and add the following environment variables:

   ```
       PORT=<give nay port number>

       DB_URI=<get from mongodb>

       FIREBASE_SERVICE_KEY=<get from firebase>

       SITE_DOMAIN=<your client url>

       STRIPE_SECRET=<get from stripe>
   ```

5. Run `npm run dev` to run the project server

The project will be available on http://localhost:5000/ by default.
